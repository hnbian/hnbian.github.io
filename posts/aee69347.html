Iterm2 快捷键记录

Iterm2
others
发布日期:   2021-04-09

1. 光标控制

快捷键 功能
光标移动到行首 ctrl + a
光标移动到行末 ctrl + e
光标后移，相当于向后方向键 ctrl + f
光标前移，相当于向前方向键 ctrl + b
上一条命令，相当于方向键上 ctrl + p
搜索命令历史 ctrl + r
删除当前字符 ctrl + d
删除之前的字符 ctrl + h
删除光标前的单词 ctrl + w
删除到文本末尾 ctrl + k
交换光标处文本 ctrl + t
将字体缩小 ⌘ + -
将字体放大 ⌘ + +
将字体改到最小 ⌘ + 0
滚到新的一屏 ⌘ + r

2. 窗口操作

快捷键 功能
新建窗口 shift + command + d（横向）command + d（竖向）
关闭窗口 shift + command + w
前一个窗口 command + `
后一个窗口 command + ~
进入窗口 1,2,3 option + command + 编号

3. 标签页操作

快捷键 功能
新建标签页 Command + T
关闭标签页 Command + W
前一个标签页 Command + 左方向键
前一个标签页 Shift + Command + [
后一个标签页 Command + 右方向键
后一个标签页 Shitf + Command + ]
进入标签页1，2，3… Command + 标签页编号

4. 面板操作

快捷键 功能
垂直分割 Command + D
水平分割 Shift + Command + D
前一个面板 Command + [
后一个面板 Command + ]
切换到上/下/左/右面板 Option + Command + 上下左右方向键

5. 其他功能

支持自定义全局快捷键用于显示和隐藏iTerm2 Preference -> Keys －> Show/hide iTerm2 with a system-wide hotkey 打上勾之后

快捷键 功能
进入和退出全屏 Command + Enter
查看当前终端中光标的位置 Command + /
命令自动补全 Command + ;
开启和关闭背景半透明 Command + u
清屏（重置当前终端） Command + r

6. 位置书签

快捷键 功能
Command + Shift + m 保存当前位
Command + Shift + j 跳回刚刚保存的位置

7. 粘贴历史

快捷键 功能
Command + Shift + h 呼出粘贴历史，支持模糊检索。还可以设置将粘贴历史保存在磁盘上（Preferences -> General）

8. 即时回放

快捷键 功能
Command + Opt + b 打开即时回放，按Esc退出。即时回放可以记录终端输出的状态，让你“穿越时间”查看终端内容。默认每个会话最多储存4MB的内容，可以在设置中更改（Preferences -> Genernal -> Instant Replay）
